# PDF parser
A ruby based lambda function that takes a base64 encoded pdf file and a ruby regular 
expression, parses the PDF file and return the text found based on the regex matches.

## Build & deploy
To build the zip file and deploy to AWS lambda manually, run
```
./build.sh
```

To build and deploy to AWS lambda automatically, given that aws cli has been set up, run
```
./build.sh -d
```

## Usage
Send a GET request to the lambda public URL with below request body:
```
{
  "file": [base64 encoded pdf],
  "regex": [regex pattern string]
}
```
